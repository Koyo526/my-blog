## Overview

This portfolio site is built using modern web technologies.
Created through `New Year's Holiday Driven Development`.

## Key Features

### Internationalization (i18n)
- Japanese/English switching using **next-intl**
- URL-based routing (`/ja/`, `/en/`)
- Content management via translation files

### Dark Mode
- Theme switching with **next-themes**
- System preference synchronization
- Manual toggle button

### Timeline Career Display
- Chronological display of work and education history
- Visual distinction by color (Work: blue, Education: green)
- Current position highlighted with pulse animation
- Scroll animations with **Framer Motion**

### Certifications & Awards Section
- Card display with links to credentials
- Links to external certification pages (Oracle, AWS Credly, etc.)

### Blog Features
- **MDX**-based Markdown article management
- Syntax highlighting (rehype-highlight)
- Automatic reading time calculation
- Tag-based categorization

### SEO Optimization
- Dynamic OG image generation
- Automatic sitemap.xml / robots.txt generation
- Optimized metadata

## Tech Stack

| Category | Technology |
|----------|------------|
| Framework | Next.js 16.1.1 (App Router) |
| UI | React 19 |
| Language | TypeScript |
| Styling | Tailwind CSS v4 |
| i18n | next-intl |
| Content | MDX (next-mdx-remote) |
| Animation | Framer Motion |
| Analytics | Vercel Analytics, Google Analytics |

## Architecture

- Fast page loads with **SSG (Static Site Generation)**
- Modern routing with **App Router**
- Component-driven design
- Type-safe validation with Zod
